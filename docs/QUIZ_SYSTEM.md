# 📚 AI Vision Lab - 答题系统使用说明

## 🎯 答题系统简介

AI Vision Lab 新增了互动答题系统，帮助中小学生通过趣味答题的方式巩固AI知识！

### ✨ 功能特点

- 📝 **20道精选题目**：涵盖AI基础、计算机视觉、YOLO算法、机器学习等知识点
- 🎚️ **三种难度级别**：
  - 简单（10分）- 基础概念
  - 中等（20分）- 应用理解  
  - 困难（30分）- 深度思考
- 👤 **用户排行榜**：实时记录成绩，激发学习竞争力
- 💡 **即时反馈**：每题都有详细解释，边答边学
- 🎨 **炫酷界面**：科技感十足的渐变UI

---

## 📖 如何使用

### 1. 进入答题系统

**方法1：侧边栏**
- 点击左侧侧边栏的 **答题** 按钮（橙色图标）

**方法2：学习完知识**
- 阅读完 [AI_LEARNING_GUIDE.md](AI_LEARNING_GUIDE.md)
- 进入答题系统测试学习成果

### 2. 开始答题

1. **输入用户名**
   - 至少2个字符
   - 会显示在排行榜上

2. **点击"开始挑战"**
   - 题目随机打乱
   - 开始计时

3. **答题流程**
   - 阅读题目和选项
   - 选择答案（A/B/C/D）
   - 点击"提交答案"
   - 查看解析
   - 点击"下一题"继续

4. **查看成绩**
   - 总分
   - 正确率
   - 答对题数
   - 用时
   - 排行榜名次

---

## 🏆 排行榜规则

### 排名依据
按**总分**从高到低排序

### 积分规则
- 简单题：10分/题
- 中等题：20分/题
- 困难题：30分/题
- **答错不扣分**

### 最高分
20题全对：20×10 + 0×20 + 0×30 = 约300-400分
（实际分数取决于题目难度分布）

### 记录保存
- 使用浏览器 localStorage 保存
- 保留前10名成绩
- 清除浏览器数据会重置

## 🎓 题目示例

### 简单题示例
**问题**：AI是什么的缩写？
- A. 人工智能 ✓
- B. 自动化
- C. 高级智能
- D. 计算机科学

**解析**：AI是Artificial Intelligence的缩写，中文意思是人工智能。

---

### 中等题示例
**问题**：置信度85%表示什么？
- A. 识别速度很快
- B. AI有85%确定这个识别结果是对的 ✓
- C. 图片质量85分
- D. 训练了85轮

**解析**：置信度表示AI对识别结果的确信程度，85%表示AI认为有85%的把握识别正确。

---

### 困难题示例
**问题**：如果一个模型在训练数据上表现很好，但在新数据上表现很差，这叫什么？
- A. 欠拟合
- B. 过拟合 ✓
- C. 正常现象
- D. 模型损坏

**解析**：过拟合（Overfitting）是指模型把训练数据记住了，但没有学到通用规律，遇到新数据就不行了。

## 📝 添加新题目

如果想添加新题目，编辑 `quizData.ts`：

```typescript
{
  id: 21,  // 新ID
  question: "你的问题？",
  options: ["选项A", "选项B", "选项C", "选项D"],
  correctAnswer: 0,  // 正确答案索引（0-3）
  explanation: "详细解释...",
  difficulty: 'medium',  // easy/medium/hard
  category: '分类名称'
}
```
