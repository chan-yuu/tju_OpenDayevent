# 🎉 AI Vision Lab v3.0 - 功能完整说明

## 📦 项目概述

**AI Vision Lab** 是一个面向中小学生的AI视觉识别教学平台，集成了完整的学习、实践、测试流程。

---

## 🎯 四大核心功能

### 1. 📊 数据标注系统

**功能描述**：标记图片中的物体，构建训练数据集

**使用流程**：
1. 将图片放入 `dataset/images/` 目录
2. 点击"启动标注工具"按钮
3. 使用 LabelImg 框选物体并选择类别
4. 标注自动保存到 `dataset/labels/`

**特点**：
- 自动启动 LabelImg
- 预定义类别列表
- YOLO格式标注
- 一键式操作

---

### 2. 🧠 模型训练系统

**功能描述**：使用标注数据训练AI模型

**使用流程**：
1. 确保已完成数据标注
2. 设置训练轮数（建议50-100）
3. 点击"开始训练"
4. 查看实时训练进度
5. 等待训练完成

**特点**：
- 实时进度显示
- 自动保存模型
- 可配置参数
- 后台训练

**训练配置**：
- 模型：YOLOv8n
- 图像大小：640×640
- 批次大小：16
- 不使用预训练权重（避免识别训练外类别）

---

### 3. 🔍 智能识别系统

**功能描述**：使用训练好的模型识别图片中的物体

**使用流程**：
1. 选择要使用的模型
2. 上传图片或启动相机
3. 点击"识别"按钮
4. 查看识别结果

**特点**：
- 支持图片上传
- 支持实时相机
- 多模型切换
- 置信度显示
- 扫描动画效果
- 自动过滤非训练类别

**识别信息**：
- 物体名称
- 边界框位置
- 置信度百分比
- 识别数量统计

---

### 4. 🎓 知识答题系统 **[NEW]**

**功能描述**：通过选择题测试AI知识掌握程度

**使用流程**：
1. 输入用户名（至少2个字符）
2. 点击"开始挑战"
3. 逐题答题
4. 查看解析
5. 完成后查看成绩和排名

**特点**：
- 20道精选题目
- 随机打乱顺序
- 三种难度级别
- 即时答案反馈
- 详细知识解析
- 实时排行榜
- 成绩记录
- 计时功能

**积分规则**：
- 简单题：10分
- 中等题：20分
- 困难题：30分
- 答错不扣分

**题目分类**：
- 基础概念（5题）
- 物体检测（5题）
- 机器学习（5题）
- YOLO算法（3题）
- 实践应用（5题）
